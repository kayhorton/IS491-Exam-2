IS491 Exam 2 App

This Exam 2 app is a the culumative of learning HTML5, JavaScript, CSS3, AJAX, using Komodo Edit, Adobe PhoneGap, and GITHUB. This app was built focusing on basic business processes and functionality using web services to interact with a database creating, modifying and deleting data.

What is HTML?

HTML stands for HyperText Markup Language. HTML is the main markup language for describing the structure of Web pages.

What is Javascript?

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

What is CSS3?

CSS stands for Cascading Style Sheet. CSS allows you to specify various style properties for a given HTML element such as colors, backgrounds, fonts etc.

What is AJAX?

Ajax (Asynchronous JavaScript and XML) is a method of building interactive applications for the Web that process user requests immediately.

What is Komodo Edit?

Komodo Edit is a free, Open-Source, multi-platform, multi-language text editor for dynamic programming languages.

What is Adobe PhoneGap?

Adobe PhoneGap is an open source framework for quickly building cross-platform mobile apps using HTML5, Javascript and CSS.

What is GitHub?

GitHub is a web-based Git repository hosting service, which offers all of the distributed revision control and source code management (SCM) functionality of Git as well as adding its own features.
